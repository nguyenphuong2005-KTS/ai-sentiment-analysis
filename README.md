# Đề tài 5: Phân Tích Cảm Xúc Phản Hồi Khách Hàng
- **Sinh viên:** Nguyễn Ngọc Bình Phương
- **Công nghệ:** Node.js, Express, Groq API (Llama 3)
- **Deployment:** Vercel & Docker

## Cách chạy dự án:
1. Copy API Key vào file `.env`.
2. Mở Terminal gõ: `npm install` để cài thư viện.
3. Chạy Server: `node server.js`.
4. Truy cập: `http://localhost:3000`.

## Cách chạy Docker:
1. `docker build -t app-ai-phuong .`
2. `docker run -p 3000:3000 app-ai-phuong`
